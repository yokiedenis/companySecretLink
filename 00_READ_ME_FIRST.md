## 🔧 CUSTOMIZATION EXAMPLES

### Change Company Name (2 seconds)
```html
<!-- Line 10 in index.html -->
<h1 class="logo">Your Company Name</h1>
```

### Change Colors (30 seconds)
```css
/* Lines 1-15 in style.css */
:root {
    --primary-color: #007bff;      ← Change this
    --secondary-color: #28a745;    ← And this
    --accent-color: #ff6b35;       ← And this
}
```

### Update Team Member (1 minute)
```html
<!-- Find "Team Section" in index.html -->
<h3>New Name</h3>
<p class="member-role">New Role</p>
<p class="member-bio">New bio...</p>
```

```html
<!-- Find "Initial Capital" in index.html -->

```
